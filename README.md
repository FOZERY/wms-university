# WMS

Система управления складом для завода по производству спецтехники

## О проекте

Веб-приложение для автоматизации учёта материалов и готовой продукции на складе производственного предприятия. Система заменяет бумажные накладные и ручные подсчёты, предоставляя цифровой инструмент для управления запасами.

Цель проекта: Курсовая работа по дисциплине "CASE-средства структурного проектирования", демонстрирующая разработку полноценной бизнес-системы.
